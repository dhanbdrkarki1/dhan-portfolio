# Portfolio Website

## Tech Stack

- Next.js 16.1.1
- React 19.2.3
- TypeScript 5.3
- Tailwind CSS 3.4
- Framer Motion 12.23
- Formspree 3.0

## Setup

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
```
